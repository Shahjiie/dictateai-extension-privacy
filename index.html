# Privacy Policy — DictateAI **Last updated:** March 25, 2026 ## Data Collection DictateAI does **not** collect, store, or transmit any personal data. ## Speech Recognition All speech recognition is performed **locally in your browser** using the built-in [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API). Audio is processed by your browser's speech engine and is never sent to any third-party server by this extension. ## Permissions | Permission | Why it's needed | |-------------|-----------------| | `activeTab` | To inject the widget into the current page | | `scripting` | To programmatically inject the content script when needed | | `storage` | Reserved for future user preferences (not currently used) | | `tabs` | To identify the active tab for keyboard shortcut handling | ## Third-Party Services This extension does **not** use any third-party analytics, tracking, or advertising services. ## Changes If this policy changes, the update will be reflected in this file and in the extension's version history. ## Contact If you have questions about this policy, please open an issue on the [GitHub repository](https://github.com/Shahjiie/dictateai-extension/issues).
